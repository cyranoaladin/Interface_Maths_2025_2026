1) Mémo — Formules essentielles

Suites arithmétiques

  • Définition : $u_{n+1} = u_n + r$ où $r$ est la raison
  • Terme général : $u_n = u_0 + nr$ ou $u_n = u_1 + (n-1)r$
  • Somme : $S_n = \frac{n+1}{2}(u_0 + u_n) = \frac{n+1}{2}(2u_0 + nr)$

Suites géométriques

  • Définition : $u_{n+1} = u_n \times q$ où $q$ est la raison
  • Terme général : $u_n = u_0 \times q^n$ ou $u_n = u_1 \times q^{n-1}$
  • Somme : $S_n = u_0 \times \frac{1-q^{n+1}}{1-q}$ si $q \neq 1$

Limites

  • Suite arithmétique : $\lim_{n \to +\infty} u_n = +\infty$ si $r > 0$, $-\infty$ si $r < 0$
  • Suite géométrique : $\lim_{n \to +\infty} u_n = +\infty$ si $|q| > 1$, $0$ si $|q| < 1$

Formule du terme général d'une suite arithmétique ?

Réponse

$u_n = u_0 + nr$ ou $u_n = u_1 + (n-1)r$

Formule de la somme des n premiers termes d'une suite géométrique ?

Réponse

$S_n = u_0 \times \frac{1-q^{n+1}}{1-q}$ si $q \neq 1$

Condition pour qu'une suite géométrique converge vers 0 ?

Réponse

$|q| < 1$
