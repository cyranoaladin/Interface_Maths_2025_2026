Alaeddine Ben Rhouma — Professeur agrégé de mathématiques

Les 3 piliers

Un écosystème complet pour les mathématiques

Outils innovants

  • Labo interactif JSXGraph
  • Rendu LaTeX natif (MathJax / KaTeX)
  • Architecture moderne (FastAPI + JS + PWA)
  • Déploiement automatisé (CI/CD GitHub Actions)

Au service des apprentissages

  • QCM interactifs auto-corrigés
  • Bilans personnalisés générés par IA (Claude)
  • Flashcards de révision
  • Progression gamifiée (niveaux + badges)
  • Mode offline (Service Worker)

Au service de l'enseignement

  • Dashboard enseignant sécurisé
  • Analytics visuels (Chart.js)
  • Import CSV → base élèves en un script
  • Réinitialisation de mots de passe en un clic
  • Bilans PDF exportables
  • Suivi groupe par groupe
